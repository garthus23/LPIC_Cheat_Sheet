Filesystem

    /bin User utility programs

    /boot files used during the system boot process

    /dev device files for the hardware devices system

    /etc System configuration files

    /home Users home folder

    /lib Stores kernel modules and shared libraries

    /lost+found Files saved during unplanned system shutdowns

    /media Temporary mount directory for removable media devices

    /mnt Temporary mount directory for removable media devices

    /opt Third application software

    /proc Virtual filesystem for running processes and kernel information

    /root Root user directory

    /run A tmpfs filesystem mounted early dur ing the boot process and used by varioustools

    /sbin Administrator utility programs

    /sys Virtual filesystem stores information con- cerning and allows modification of con- nected hardware devices.

    /tmp Temporary files

    /usr Stores binaries, documentation, source code, libraries, and so on

    /var Contains log, lock, spool, mail, and temp files

Soft Monitor

    free Shows the amount of free/used physical and swap memory

    top Multiple display panels that show various resource usage data such as processes consuming the most CPU

    htop Enhancement of the top utility

    iostat Shows device I/O loading summary broken down per device

    iotop Similar to the top utility, it shows current I/O usage by processes (or threads)

    mpstat Shows multiple processor statistics

    pmap Shows a processes map for the PID parameter

    ps Shows current process information, including CPU consumption

    pstree Shows current processes in a tree format

    sar Acronym for System Activity Reporter: a Static or Dynamic multiple resource monitoring utility

    uptime Shows how long the system has gone without a reboot

    vmstat Shows swap (virtual memory) performance

    w Shows current user information, including CPU consumption.

Net Monitor

    iftop shows current network traffic information, including DNS

    ip display network and routing statistics

    iptraf Shows network information, and it is menu driven.

    lsof -p $PID Shows open files and network connections by process

    mtr Shows routing information for the URL parameter

    ntop Gathers network statistics that can be viewed via a web browser via port 3000

    ss Displays socket statistics directly from kernel space

    tcpdump A packet analyzer/sniffer
